# 🚦 Road Accident Analysis Dashboard

## 1. Project Title / Headline

🚗 **Road Accident Insights: Interactive Traffic Accident Analysis Dashboard**

An interactive **Power BI dashboard** designed to analyze road accident patterns by vehicle type, location, and accident distribution. The dashboard enables users to explore accident trends, understand high-risk vehicle categories, and identify critical road safety insights through dynamic visualizations.

---

# 2. Short Description / Purpose

The **Road Accident Analysis Dashboard** is an interactive Power BI report created to analyze traffic accident data across different vehicle types and regions. It helps users explore accident patterns, identify high-risk vehicles, and understand trends that can support road safety improvements and policy decisions.

This dashboard can be used by **transport authorities, safety analysts, urban planners, and researchers** to gain insights into accident frequency and contributing factors.

---

# 3. Tech Stack

The dashboard was built using the following tools and technologies:

* **📊 Power BI Desktop** – Main platform used to design and develop the interactive dashboard
* **📂 Power Query** – Used for data cleaning, transformation, and preprocessing
* **🧠 DAX (Data Analysis Expressions)** – Implemented to create calculated measures and KPIs

* **📁 File Format** – `.pbix` used for development and visualization design

# 4. Data Source

The dataset used in this project contains **road accident data categorized by vehicle type and location**.

The data includes information such as:

* Vehicle types involved in accidents (Car, Bike, Bus, Van, Tractor, Others)
* Accident distribution by location
* Accident counts and patterns
* Geographic accident mapping
* Traffic-related statistics for analysis

The dataset was processed and transformed using **Power Query** before being visualized in Power BI.

---

# 5. Features / Highlights

• Business Problem

Road accidents are one of the leading causes of injuries and fatalities worldwide. Transportation departments collect large volumes of accident data, but analyzing raw datasets makes it difficult to quickly identify patterns, high-risk vehicle types, and accident-prone locations.

Key questions such as:

• Which vehicle types are involved in the highest number of accidents?
• What regions record the largest share of accidents?
• How are accidents distributed across different transportation categories?
• Which vehicle types contribute the most to road safety risks?

# Goal of the Dashboard

To deliver an interactive visual analytics tool that:

• Enables users to explore accident data across 6 major vehicle categories
• Helps identify high-risk vehicles and accident distribution patterns
• Supports data-driven decisions for road safety planning
• Provides insights that help authorities and planners reduce accidents and improve road safety
